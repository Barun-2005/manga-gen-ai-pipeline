# Manga Generation Project

This project utilizes ComfyUI for generating manga-style images using AI models. The folder structure is organized to separate different types of models and assets.

## Folder Structure

- `assets/negatives`: Contains negative prompt images.
- `models/loras`: Stores LoRA models.
- `models/checkpoints`: Contains model checkpoints.
- `models/controlnet`: Stores ControlNet models.
- `models/ipadapter`: Contains IP Adapter models.
- `models/vae`: Stores VAE models.
- `extensions`: Contains project-specific extensions.
